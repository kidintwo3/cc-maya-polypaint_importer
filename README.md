Polypaint Importer for Maya

Usage (Python command):
----------------------

- Select object from zbrush
- Type in the Python command (name of the source obj):

	import maya.cmds as cmds;
	cmds.polyPaintImporter("D:/vertcol.OBJ", gamma=2.2);
